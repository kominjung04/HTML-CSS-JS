일반적인 글꼴의 기본 크기

1. 절대적인 크기 pt, px, cm, mm, in, pc, Q

  • pt :: 1pt = 1in의 1/72
  • px :: 1px = 1in의 1/96
  • cm :: 1cm = 96px/2.54
  • mm :: 1mm = 1cm의 1/10
  • Q :: 1Q = 1cm의 1/40
  • in :: 1in = 2.54cm = 96px
  • pc :: 1pc = 1in의 1/16

2. 상대적인 크기 em, %,rem

em, %는 상대적 크기, 부모 요소 크기에 따라 유연하게 변하는 반응형의 단위

font size( 14px )
font size (14 * 1.2 = 16.8px)
font size(16.8 * 1.2 = 20.16px)
font size(20.16 * 1.2 = 24.192PX)

rem(root em) 항상 html 요소를 기준으로 계산됨

Parents
Child(32px) 부모태그 상관없이 html태그만 기준

3. 각도 단위 deg, grad, rad, turn

  • deg :: 도를 지칭, 360도가 완전한 원
  • grad :: 그라디안을 지칭, 400 그라디안이 완전한 원
  • rad :: 라디안을 지칭, 2π 라디안이 완전한 원
  • turn :: 턴을 지칭, 1턴이 완전한 원

4. 시간 단위 s, ms

  • s :: 초
  • ms :: 밀리초, 1000ms = 1s

5. 해상도 단위 dpi, dpcm, dppx, x

  • dpi :: 인치당 점의 수.
  • dpcm :: 센티미터당 점의 수.
  • dppx, x :: 픽셀당 점의 수.

6. 모바일 단위 vw, vh

  • vw :: 뷰포트 너비의 1%.
  • vh :: 뷰포트 높이의 1%.
